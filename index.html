Git Commands



Config le nom et l'adresse mail

            git config --global user.name "nom" 
            
            Copié !
            git config --global user.email "adresse"
            
            Copié !

Associer le dépot local avec le dépot distant

            git remote add origin (adresse)
            
            Copié !

Modifier les N derniers commits

            git rebase -i HEAD~N
            
            Copié !



les commits 

        

        
feat : commit qui ajoute une nouvelle fonctionnalité 

        
fix : commit qui corrige un bug

        
docs : commit qui ajoute ou modifie de la documentation

        
style : commit qui n'impacte pas le code (espace, indentation, point-virgule manquant, etc)

        
refactor : commit qui ne corrige pas de bug ni n'ajoute de fonctionnalité

        
perf : commit qui améliore les performances

        
test : commit qui ajoute ou modifie des tests

les branches 

        

            
master : branche principale

            
develop : branche de développement

            
feature : branche de fonctionnalité

            
bugfix : branche de correction de bug

            
hotfix : branche de correction de bug mais dans l'urgence

            
test : branche pour essayer des choses

Créer une nouvelle branche

            git checkout -b (nom_branche)
            
            Copié !

Changer de branche

            git checkout (nom_branche)
            
            Copié !

Supprimer une branche local

            git branch -D (nom_branche)
            
            Copié !

Supprimer une branche distante

            git push --delete origin (nom_branche)
            
            Copié !

Merger une branche

            git merge (nom_branche)
            
            Copié !
